# Changelog

## v0.4.1 — 2026-04-27

直感操作 UX の刷新（オンボーディング・キーボード・D&D・右クリック）、ガント図のタスクバー直接操作、SVG 検索ハイライト、SVG クリック接続モードの不具合修正、XY チャート Y 軸の自動追従を含む累積リリース。

### 新機能 — 直感操作 UX 基盤（全 21 図エディタ）

- **初回オンボーディングヒント** — 各エディタを初めて開いた際に使い方ガイドを表示。「次回から表示しない」を `localStorage` に保存。
- **キーボードショートカット統一** — `Delete` 削除 / `↑↓` 並び替え / `Enter` 編集 / `Ctrl+Z`・`Ctrl+Y` 元に戻す/やり直し（フォーム系 14 種）。
- **ツールチップ強化** — 主要ボタンに「機能名 + ショートカット」のツールチップを付与。
- **レスポンシブレイアウト** — `@media (max-width: 900px / 600px)` で左パネルと SVG エリアを縦積みに切替。
- **リスト項目のドラッグ&ドロップ並び替え** — 14 種フォーム系エディタ（状態遷移・パイ・ジャーニー・GitGraph・タイムライン・要求図・C4・Sankey・XY・Block・ZenUML・Packet・Architecture・Kanban）で左パネルの項目を掴んで並べ替え可能。半透明・青ラインの視覚フィードバック付き。
- **右クリックコンテキストメニュー（共通基盤）** — VS Code テーマ準拠の軽量コンテキストメニュー `DiagramCommon.showContextMenu()` を追加。Esc / 外部クリック / スクロールで自動閉鎖。

### — 各図種の SVG 上右クリックメニュー（21 種統一）

- **フローチャート** — ノード上で `ラベル編集 / 色変更 / ここから接続 / 削除`、エッジ上で `編集 / 線種切替 / 削除`。
- **シーケンス図** — 参加者上で `編集 / 削除`、メッセージ線上で `編集 / ↑↓ 順序変更 / 削除`。
- **クラス図** — クラスノード上で `クラス名編集 / 属性追加 / メソッド追加 / 削除`、リレーション上で `編集 / 削除`。
- **象限チャート** — ポイント上で `編集 / 削除`。
- **マインドマップ** — ノード上で `子ノード追加 / 名前を編集 / このノードを削除`。
- **ER 図** — エンティティ上で `名前を変更 / 属性を追加 / このエンティティを削除`。
- **フォーム系 14 種のリスト項目** — `編集 / ↑上に移動 / ↓下に移動 / 🗑 削除`。

### — ガント図のタスク直接操作

- **タスクバーのドラッグで開始日変更** — SVG 上のタスクを掴んで左右にドラッグすると、ピクセル/日換算で `startDate` が更新され、コードに即時反映。
- **タスクバー右端のドラッグで期間変更** — バーの右端 8px 領域でカーソルが `ew-resize` に変化し、ドラッグで `duration` を 1 日単位で増減。
- **タスクバーの右クリックメニュー** — `±1日 / ±7日` の即時シフトと `削除`。`after X` 依存タスクはドラッグ不可（`not-allowed` カーソル）。
- ピクセル/日比は描画後の各バー幅 ÷ 期間の平均から自動キャリブレーション。

### 改善 — 検索ハイライト

- **SVG 図中の文字も検索ヒットで強調表示** — フローチャート/シーケンス図など Mermaid 図のラベルが検索語を含む場合、`<tspan>` 分割により該当文字列を黄色の縁取りで強調。
- **SVG 内テキストが消えて見える不具合を修正** — 検索ハイライトで SVG 内のテキストノードを HTML の `<mark>` で包んでしまい、文字が表示されなくなる問題を解消。

### — XY チャート Y 軸の自動追従

- **データ値が Y 軸の上限を超えても表示が振り切れる問題を解消** — `state.yAuto`（既定 ON）を導入し、`_generate()` 時に全シリーズ値からキリの良い最小・最大を算出して `y-axis` に反映。Y 軸ダイアログから「データに自動追従: ON / OFF」を切替可能（OFF 時は従来通り手動範囲を尊重）。ソースに既存の `y-axis "..." min --> max` がある場合は手動指定として保持。

### 修正 — SVG クリック接続モードのハイライト範囲

- **状態遷移図 / 要求図** — 「遷移追加」「リレーション追加」モードで図全体が光って選択中のノードが分からなくなる不具合を修正。共通基盤 `_maybeAttachSvgConnect` がノード `<g>` の祖先（=図全体を包む `<g>`）まで一致対象にしていたため、最も内側に一致する `<g>` のみを対象とするフィルタを追加。`<title>` テキストもマッチ対象に含めて検出精度を改善。
- **アーキテクチャ図** — サービス／グループ単位で接続モードのクリックが効かなかった不具合を修正。`ArchitectureEditor._attachSvgConnectHandlers` に同等の innermost フィルタを適用し、さらに mermaid が出力する `data-id` / `id` 属性（末尾 `-svc1` 形式を含む）と `<title>` テキストによるノード判定を追加。

### 内部

- `DiagramCommon` ネームスペース（`mountOnboarding` / `attachKeyHandlers` / `tooltip` / `showContextMenu` / `enableDragSort`）を整備。
- `editor.css` に `.dve-onboarding*` / `.dve-ctxmenu*` / `.dve-draggable*` / `.dve-list-item:focus` などの共通スタイルを追加。

## v0.3.x — 2026-04-18

Mermaid 11.14 へのアップグレードと、対応 21 図種化、および拡張機能要望（テーマ切替・検索置換・リンク Ctrl+Click 等）の累積リリース。

### 新機能 — Mermaid 全種対応（v0.3.0）

- **Mermaid 11.14 へアップグレード**（10.6 → 11.14）— `packet-beta` / `architecture-beta` / `kanban` を含む最新ダイアグラムを描画可能。
- **GUI 編集対応を 7 種 → 21 種に拡張** — 既存 7 種に加え、状態遷移図 / パイチャート / ユーザージャーニー / Git グラフ / タイムライン / 要求図 / C4 図 / Sankey 図 / XY チャート / ブロック図 / ZenUML / パケット図 / アーキテクチャ図 / Kanban の 14 種を新規対応。
- **Mermaid ピッカーの拡張** — `Mermaid` ツールバーボタンから 21 種のテンプレートを選択して挿入可能。
- 新規モジュール `media/extra-diagram-editors.js` に汎用フォーム型エディタ基底クラス `GenericFormDiagramEditor` と各派生クラスを実装。検出ヘルパー `window.ExtraDiagramUtils` を追加し既存ルーティング (`startMermaidEditing`) から 14 種を分岐。

### — 拡張機能要望対応（v0.3.1）

- **ライト/ダークテーマ切替ボタン（☀️/🌙）** — ツールバーから WebView 全体のテーマをワンクリックで切替。Mermaid のテーマも追従して再描画。状態は `vscode.setState` で保持。
- **リンクの Ctrl/Cmd+Click 遷移** — プレビュー中のリンクを `Ctrl+Click`（macOS は `Cmd+Click`）で開けるように。`http(s)/mailto` は外部ブラウザ、相対パスはドキュメント基準で VS Code が開く。
- **Ctrl+F 検索 / Ctrl+H 置換** — 専用 Find/Replace バーを追加。大小区別・正規表現・前後ナビゲーション・「置換」「すべて置換」をサポート。検索結果はプレビュー上にハイライト表示。
- **テキストエディタへの切替ボタン（📝）** — ワンクリックで同じファイルを VS Code 標準テキストエディタで開く。
- **Mermaid フローチャート — レイアウト選択** — ツールバーに Dagre / ELK / ELK ツリー の切替を追加。`%%{init:{"layout":"..."}}%%` ディレクティブとして保存。
- **Mermaid フローチャート — グループの入れ子化** — 既存サブグラフを別サブグラフ配下にネスト可能に。グループ一覧から「親グループ」セレクタで親変更、複数選択チェック → 「複数グループを 1 つに結合」で一括ネスト（サイクル防止付き）。

### 改善

- 編集モードのブロック背景を透過に変更（編集前後で色変化を抑制）。
- Mermaid ビジュアルエディタの初期表示で、図がエリアに収まる場合は自動フィットせず 100% を保持。エリアを超える場合のみ自動フィット。

### 修正（v0.3.2）

- Mermaid レンダリング時に DOM へ残る孤立 SVG（爆弾アイコン / 「Syntax error in text」）を周期的に除去するクリーンアップを追加（`cleanupMermaidOrphans` + `MutationObserver`）。
- 要求図 (`requirementDiagram`) で日本語値を含むケースを自動的に `"…"` で囲んで生成。
- ZenUML を図種ピッカーから除外（UMD バンドルに含まれていないため）。
- 画面幅を全体使用（`#editor-container` を full-width 化）、`</div>` 欠落で全体が非表示になる不具合を修正。

### 内部

- `MarkdownVisualEditorProvider` に `openLink` / `openAsText` メッセージハンドラ追加。
- `FlowchartModel` に `layout` / `parentSgId` を追加し、`generate()` がネストされた `subgraph` を再帰出力。`parse()` が init ディレクティブと入れ子構造を読み取り。Undo スナップショットに `layout` を含める。

## v0.2.1 — 2026-04-11

### 新機能

- **ツールバー挿入位置ピッカー** — ブロック未編集時にツールバーボタンを押すと、挿入位置選択ダイアログが表示され「先頭に挿入」または任意のブロックの後に挿入可能。
- **Mermaid ダイアグラム種別ピッカー** — Mermaid ボタンから 7 種類のダイアグラム（フローチャート・シーケンス図・クラス図・マインドマップ・ER図・ガントチャート・象限チャート）を選択して挿入。
- **ガントチャート — セクション折りたたみ** — ▼/▶ トグルボタンでセクションを展開/折りたたみ。折りたたみ時はタスク数を表示。
- **ガントチャート — ドラッグ＆ドロップ並び替え** — タスク行のドラッグハンドル（≡）で同一セクション内・セクション間のタスク移動に対応。
- **ガントチャート — 色カスタマイズ** — セクション背景色を設定すると配下タスクに自動適用（SVG 上）。タスク個別の背景色で上書きも可能。18 色のプリセット + カスタム色入力。

### 改善

- ガントチャートのリストパネル幅を 40%（最大 600px）に拡張。
- ダイアグラムエディタの左サイドバーレイアウト対応。
